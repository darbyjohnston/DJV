Opening Files

Icon Menu Item Description Keyboard Shortcut
Open Open a new file Ctrl O
Recent List of recently opened files
Reload Reload the current file Ctrl R
Reload Frame Reload the current frame Shift Ctrl R
Close Close the current file Ctrl W

Files may also be opened by either dragging and dropping them onto the image view, or by specifying them on the command-line.

Proxy Scale

Proxy scaling reduces the resolution of images as they are loaded. This can allow more images to fit in the memory cache at the expense of image quality.

Menu Item Description
Proxy Scale/None Disable proxy scaling
Proxy Scale/1/2 Reduce the resolution of loaded images by 1/2
Proxy Scale/1/4 Reduce the resolution of loaded images by 1/4
Proxy Scale/1/8 Reduce the resolution of loaded images by 1/8

8-bit Conversion

Convert images to 8-bits as the are loaded. This can allow more images to fit in the memory cache at the expense of image quality.

Menu Item Description
8-bit Conversion Toggle whether 8-bit conversion is enabled

Memory Cache

The memory cache stores images for faster playback performance.

Menu Item Description
Memory Cache Toggle whether the memory cache is enabled
Preload Cache Load frames automatically when playback is stopped
Clear Cache Toggle whether the memory cache is enabled
