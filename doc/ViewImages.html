Frame Store

The frame store keeps a copy of an image that you can use to compare with the current frame.

Menu Item Description Keyboard Shortcut
Show Frame Store Toggle whether the frame store is visible E
Set Frame Store Copy the current image to the frame store Shift E

Transforms

Transforms can mirror, scale, and rotate the image.

Menu Item Description
Mirror Horizontal Mirror the image horizontally
Mirror Vertical Mirror the image vertically
Scale Scale the image
Rotate Rotate the image

Color Profiles

Color profiles apply a color transformation to the image to allow it to be displayed properly.

Menu Item Description Keyboard Shortcut
Color Profile Toggle whether the color profile is enabled

Display Profiles

Display profiles apply a color transformation to the image to calibrate it for individual displays.

Menu Item Description Keyboard Shortcut
Display Profile Editor Toggle whether the display profile editor is visible
Display Profile List of display profiles.
Display Profile / Reset Reset the display profile Shift `

Channels

Show individual image channels.

Menu Item Description Keyboard Shortcut
Channel / Default Show the default channels `
Channel / Red Show the red channel R
Channel / Green Show the green channel G
Channel / Blue Show the blue channel B
Channel / Alpha Show the alpha channel A
