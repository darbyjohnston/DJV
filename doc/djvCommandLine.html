Command Line

  • OpenGL Options
  • General Options

OpenGL Options
-render_filter (minify) (magnify)Set the render filter. Options = Nearest, Linear, Box, Triangle, Bell, B-Spline, Lanczos3, Cubic, Mitchell. Default = Linear, Nearest.
-render_filter_highSet the render filter to high quality settings (Lanczos3, Mitchell).

General Options
-time_units (value)Set the time units. Options = Timecode, Frames. Default = Frames.
-global_speed (value)Set the global speed. Options = 1, 3, 6, 12, 15, 16, 18, 23.98, 24, 25, 29.97, 30, 50, 59.94, 60, 120. Default = 24.
-help, -hShow the help message.
-infoShow the information message.
-aboutShow the about message.
